AR Model Viewer

How to Use This App

This AR web application allows you to view 3D models in augmented reality on your mobile device.

Visit the Library to browse all available AR models.

Adding Your Own Models

To add your own models to this application:

  1. Fork the GitHub repository
  2. Add your .glb and .usdz files to the models directory
  3. Update the models.json file with your model information
  4. Create a pull request or commit directly to your repository

For detailed instructions, see the README.md file in the repository.
